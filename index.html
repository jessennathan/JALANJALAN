Cara Publish ke GitHub Pages 🚀

Ikutin langkah ini biar website itinerary kamu bisa dibuka pakai link.

Buka GitHub, login, lalu klik New Repository.
Beri nama repo, misalnya koi-summer-trip, pilih Public, lalu klik Create repository.
Upload file HTML ini, lalu ubah nama filenya jadi index.html.
Klik Commit changes.
Masuk ke Settings → Pages.
Pada bagian Branch, pilih main dan folder /root, lalu klik Save.
Tunggu 1–3 menit. Link website kamu biasanya jadi: https://username.github.io/koi-summer-trip/
